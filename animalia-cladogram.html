암기 도우미
분류 순서 (기저→후구)
해면 → 자포 → 편형 → 연체 → 환형 → 선형 → 절지 → 극피 → 척삭
"해자편연환 선절극척"
선구 vs 후구
선구 = 원구 → 입 / 후구 = 원구 → 항문
"선(먼저)입, 후(나중)항" — 먼저 만들어진 구멍이 입이면 선구, 항문이면 후구
촉수담륜 vs 탈피
촉수담륜 = 편 · 연 · 환 / 탈피 = 선 · 절
"편연환은 담(배에 담고), 선절은 탈(옷을 벗는다)"
